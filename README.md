# Sistema de Cadastro de Alunos

Projeto desenvolvido em React para a disciplina de Frontend.

## Tecnologias Utilizadas

* React
* Vite
* Docker
* Git e GitHub

## Como executar o projeto

### Subir a aplicação

```bash
docker compose up -d --build
```

### Acessar no navegador

```txt
http://localhost:5173
```
